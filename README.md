## Installation

Windows based
- JavaScript (Vanilla)
- Node.js
- Express
- MongoDB Atlas
- Swagger UI

Dependencies requirements:

```sh

npm i nodemon --save-dev
npm i express --save
npm i cors
npm i dotenv
npm i mongoose
npm i swagger-ui-express

npm run dev //from root folder to running Nodemon server

```


## Usage

API for login access with database (MongoDB Atlas).
